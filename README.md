# WinLogWatcher

A lightweight Windows log monitoring tool that watches real-time event viewer logs, updates and timestamps each event into fast readable text file.

## Features
- Real-time log file monitoring
- Timestamp tagging
- Custom logger integration

## Usage
Compile the code using Visual Studio community 2022 (Microsoft Visual C++ 2022).
Run the application and it will automatically detect changes in the specified log file.
